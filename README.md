# GDDrive
Cloud Storage Implemented in Geometry Dash

Video: https://youtu.be/oENqzFJ3TgI

--------------

This mod does **NOT!** steal your Geometry Dash username or password! When you enter your username and password, it only stores them locally, with the password being converted to GJP2 so it isn't stored as plain text.
